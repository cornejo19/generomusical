HTML

HTML son las siglas de Hypertext Markup Languaje, que puede traducirse como lenguaje de marcas o marcado de hipertexto.
El lenguaje HTML se emplea para crear las paginas web. Es muy facil de ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otrade un mismo navegador, pero suelen tener un nombre similar.
HTML se compone de etiquetasque se escriben entre los simbolos meyor y menor que.
